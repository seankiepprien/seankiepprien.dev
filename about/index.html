About me

My coding journey started in 2016, creating modifications for an old video game called Diablo 2. After some time, i decided to learn programming using the Game Maker language (GML) and created two video games mockups of Asteroids and Tetris.
In 2020, i started learning how the Web works and began making simple websites using plain HTML and over time, expanding my knowledge by adding CSS and Javascript and it was at that time that i completely fell in love with web developpement. To this day, i helped creating many web applications with the help of Vue

HTML5
